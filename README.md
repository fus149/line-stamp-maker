# LINEスタンプ自動でつくるくん

ペット画像8枚からLINE Creators Market用スタンプを自動生成するツール。

## セットアップ

```bash
# Python 3.11推奨
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 使い方

### 1. 画像を配置

`input_images/` フォルダにペット画像を8枚配置してください。

対応形式: JPG, PNG, WebP, HEIC, BMP

### 2. 実行

```bash
python main.py
```

### 3. メッセージモードを選択

| モード | 説明 |
|--------|------|
| A | テンプレートからランダムに8個選択 |
| B | テンプレート一覧から好きな8個を選択 |
| C | 8つのメッセージを自由入力 |
| D | メッセージなし（画像のみ） |

### 4. 出力を確認

```
output/
├── 01.png 〜 08.png   # スタンプ画像（370×320px, 透過PNG）
├── preview.png         # 8枚並べた確認画像
└── line_stamp.zip      # アップロード用ZIP
```

## 画像処理の流れ

1. **画像確認** - 8枚の画像を検出、暗さ・向き補正
2. **背景切り抜き** - ペットのみ切り抜き（毛並み保持、完全透過）
3. **構図調整** - 被写体中央配置、安全余白確保
4. **文字デザイン** - 丸ゴシック・白文字・黒縁取り（顔を隠さない配置）
5. **品質チェック** - 透過・サイズ・画質の自動検証

## カスタムフォント

`.env` ファイルでフォントパスを指定可能:

```bash
cp .env.example .env
# FONT_PATH を編集
```

## 出力仕様

- フォーマット: PNG（透過）
- サイズ: 370 × 320 px
- LINE Creators Market の静止画スタンプ規格に準拠
